This introduction will not be one of those you are likely accustomed to. I will not attempt to convince you that Rust is amazing, or that it is the greatest programming language ever created from every perspective, because it simply is. Therefore, I will not waste time on frivolities and we will get straight to the point, even in the introduction. This approach will be kept throughout the book.

<Warning>
This book is not actually suited for just anyone. In general, it is intended for programmers with prior experience in the magical world of computers. Nevertheless, nothing prevents anyone from attempting to read this tome to the very end.
</Warning>

>In Rust, the compiler plays a gatekeeper role by refusing to compile code with elusive bugs, including concurrency bugs.

>Rust also brings contemporary developer tools to the systems programming world:

>- Cargo, the included dependency manager and build tool, makes adding, compiling, and managing dependencies painless and consistent across the Rust ecosystem.
>- The `rustfmt` formatting tool ensures a consistent coding style across developers.
>- The Rust Language Server powers integrated development environment (IDE) integration for code completion and inline error messages.

Ensure you have downloaded Cursor [[cursor.com](https://cursor.com)] and installed the `rust-analyzer` plugin to practice. Both have become the inevitable, de-facto standard for programming in Rust.
